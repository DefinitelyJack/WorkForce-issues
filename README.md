# Workforce - Official Issue Tracker

Welcome to the official support repository for **Workforce**, the profession management mod for Minecraft.

> [!IMPORTANT]
> **Source Code Notice:** This project is currently **closed-source**. This repository does not contain the mod's source code; it is used exclusively for issue tracking, documentation, and feature requests.

---

## Bug Reports & Support
If you've encountered a bug or a server crash, please help me fix it!

1. Go to the [**Issues**](https://github.com/TUO_USER/Workforce-Issues/issues) tab.
2. Click on **New Issue**.
3. Provide as much detail as possible:
   - **Minecraft & Forge version** (e.g., 1.20.1 - 47.2.0).
   - **Description** of what happened.
   - **Logs:** Upload your `latest.log` or `debug.log` (use [Pastebin](https://pastebin.com) if the file is too large).

## Feature Requests
Have an idea to make Workforce better? 
Open an issue and tag it as a **"Suggestion"**. I'm always looking for ways to improve the roleplay experience!
